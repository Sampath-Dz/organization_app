# Core Service - Organization Application


The Core Service is responsible for managing the core organizational data:
- Organization management
- Team management
- Membership management

- `/core/v1/organizations` - CRUD operations for organizations
- `/core/v1/teams` - CRUD operations for teams
- `/core/v1/members` - CRUD operations for members
